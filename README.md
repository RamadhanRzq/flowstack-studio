# Ramadhan Portfolio Vite

Landing page jasa pembuatan website, aplikasi mobile, backend, dan automasi operasional untuk personal brand fullstack developer.

## Stack

- Vite + React JavaScript
- Tailwind CSS v4 via `@tailwindcss/vite`
- Lucide React icons

## Menjalankan Project

```bash
npm install
npm run dev
```

Kontak dan identitas utama bisa diganti dari konstanta `profile` di `src/App.jsx`.
